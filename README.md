#  Focus Foundry
  Focus Foundry is a comprehensive web-based productivity and study management application. It is designed to enhance focus and organize learning activities through a unified, interactive dashboard.      Key Features:    Pomodoro Timer: The application includes a built-in timer that alternates between 25-minute focus sessions and 5-minute breaks. It features a visual, animated progress ring and utilizes audio alarms to signal the end of a session.    Task Management: A "Quick Tasks" section allows users to add new to-do items. Users can check off completed tasks, delete them, and filter the view to see all, pending, or completed items.    Subject Planner & Quiz Manager: Users can add and manage specific study subjects or courses. For each subject, users have the ability to create, edit, and take custom multiple-choice quizzes. The app automatically calculates and displays the user's score after finishing a quiz.    Progress Tracking: The dashboard tracks and displays the user's total study time. It also counts the number of completed Pomodoro sessions and shows the ratio of completed tasks.    Daily Motivation: A dedicated motivation section displays random inspirational quotes to encourage the user.    Technical Details:    Bilingual Support: The application is fully translated into both English and Arabic. The layout automatically shifts to a Right-to-Left (RTL) format when the Arabic language is toggled.    Customizable UI: Users can toggle between modern light and dark themes. The styling is fully responsive, adapting to both desktop and mobile screens.    Local Storage: All personal data, including tasks, courses, custom questions, and study statistics, are saved directly in the browser's local storage.
